Real-time Simulation Module

C++ | Multithreading

C++ simulation module for dynamic systems: implementation of multithreading and sensor models (IMU/GNSS).

I designed and implemented a real-time–capable simulation module in modern C++ (C++17), focusing on deterministic timing and robust multithreading. The module includes:

  • A modular architecture for dynamic system models
  • High-frequency update loops with controlled timing behaviour
  • Thread-safe sensor pipelines (IMU, GNSS) with realistic noise/error injection
  • Fault-handling logic and recovery routines
  • Minimal-latency data transfer between threads

The module is structured around deterministic simulation loops, parallel execution, and clear interfaces for extending physical models and sensor pipelines. The results are automatically saved in output/<scenario_name>
